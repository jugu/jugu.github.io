git init
Initializes the git repository in the current project folder

git add < filename >
Add the untracked file to the the staging area of git

git add .
Add all the untracked files (modified/added)

git add --all
Add all the untracked files (modified/added/DELETED)

git status
Check the status of the files (staged/unstaged)

git commit -m ""
Commit the staged/tracked files to the git repository

git config --global user.email ""
Set default identity for the account

git config --global user.name ""
Set default identity for the account

git log
Check the log for git commits related to the project

git reset HEAD < filename >
Remove the file from the staged area back to the untracked area

git checkout < filename >
Checkout the file from the previously committed version in git

git rm < filename >
Remove file from git project folder and moves the file to staging area

git checkout < committed log hash >
Checkout HEAD detached branch from a previously logged commit instance using the logged hashcode

git branch
List the branches for the project and determine the active one

git branch < newbranch >
Create a new branch using branch code (from the HEAD detached branch)

git checkout < branchname >
Checkout a branch from git to work on

git merge < branchname >
Merge the changes in the listed branch into the current active branch

git branch -m < orig-branch-name > < new-branch-name >
Rename the branch

git branch -D < branchname >
Delete the branch

git branch -a
List all the branches in github project

git clone < git url >
Clone the hosted repository

git checkout -b < customname > origin/< branchname >
Checkout remote branch from Github and move it to custom branch and makes the custom branch active

git clone --mirror < repository link .git> .git
Clone into a bare repository

git config --bool core.bare false
Convert bare repository into regular repository

git reset --hard
Grab everything inside .git folder and create all the branches present in it (Useful for downloading all github repository branches in one go)
